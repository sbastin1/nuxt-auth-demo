## Setup

```bash
#optional, select node version
nvm use 24.10

# Install Dependencies
pnpm install

cp .env.example .env

# Start Dev Server
pnpm dev
```

#### Fill out the .env file.

1. `NUXT_SESSION_PASSWORD` will be autogenerated on startup
2. Switch out `<PROVIDER>` with an OAuth Provider of your choice. eg. `GITHUB`
3. Fill out the Client Id and Secret with the data provided by your provider
4. Restart the dev server

```ru
NUXT_SESSION_PASSWORD=
NUXT_OAUTH_<PROVIDER>_CLIENT_ID=
NUXT_OAUTH_<PROVIDER>_CLIENT_SECRET=
```
